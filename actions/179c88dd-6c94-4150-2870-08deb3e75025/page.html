步数修改

Windows 桌面工具：登录 Zepp / 小米运动账号，提交并修改当日步数。由 Python 脚本迁移为 WPF 图形界面，通过 Quicker 子程序调用。

主要功能

  • 账号登录：支持手机号或邮箱登录 Zepp 账号
  • 修改步数：向 Zepp 服务器提交指定当日步数
  • 记住凭据：账号与密码经 Windows DPAPI 加密，保存在本机 %AppData%\ZeppStepTool\
  • Quicker 集成：作为子程序包发布，可在组合动作中一键唤出

环境要求

  • Windows 10 及以上
  • .NET Framework 4.7.2 或更高（首次运行若缺失，请按系统提示安装）
  • Quicker 1.45.0 及以上（与本动作发布版本一致）

使用说明

  1. 在 Quicker 面板安装本动作后，按你配置的快捷键或方式触发
  2. 在弹出窗口中输入账号、密码与目标步数，确认提交
  3. 勾选「记住密码」后，下次可免去重复输入（凭据仅保存在本机）
  4. 提交成功后，可在 Zepp / 小米运动 App 中刷新查看步数是否更新（同步可能有延迟）

常见问题

  • 无法启动：请确认已安装 .NET Framework 4.7.2+
  • 登录失败：检查账号密码、网络；部分账号需使用注册时的登录方式（手机/邮箱）
  • 步数未变化：稍等片刻后刷新 App；若仍无效，可在动作页「讨论」区反馈
