md 提示

基于 WPF + WebView2 + React 的 Markdown 提示窗口子程序：在组合动作中传入 Markdown 正文即可弹出排版良好的只读提示，支持 GFM 扩展语法与可选底部按钮。

主要功能

  • Markdown 渲染：使用 react-markdown + remark-gfm，支持表格、任务列表、删除线、代码块等 GFM 语法
  • GitHub 风格排版：内置 github-markdown-css，阅读体验接近 GitHub 文档页
  • 深浅色自适应：页面主题跟随系统 prefers-color-scheme，暗色环境下同样清晰
  • 可选底部按钮：语法与 MessageBox3md 自定义按钮一致，点击后通过子程序返回值带回 Data 字段
  • 快捷关闭：按 Esc 可关闭窗口（无按钮时同样有效）

子程序说明

本动作发布 WebViewMarkdownTip 子程序（当前版本 1.1.0）。在其它组合动作中可直接调用子程序，或使用本动作的示例流程。

托管入口（C#）：Runner.ShowMarkdownTip(markdown, title, buttonDefinitions);

markdown：要展示的 Markdown 正文（必填）

title：窗口标题，省略时为「提示」

buttonDefinitions：可选，字符串列表；每行一个按钮，格式与 MessageBox3md 自定义按钮相同，例如 确认(_S)|Ok、取消|Cancel

使用说明

  1. 安装本动作后，子程序 WebViewMarkdownTip 会一并安装到 Quicker
  2. 在组合动作中用「运行子程序」传入 Markdown 文本与标题；需要用户确认时，再传入按钮定义列表
  3. 窗口为非模态弹出，可调整大小；正文区域可滚动浏览长文档
  4. 首次使用若提示 WebView2 相关错误，请安装 Microsoft Edge WebView2 运行时

适用场景

操作结果说明 · 富文本帮助 · 带格式的确认对话框 · 展示 API 返回的 Markdown 文档 · 替代纯文本提示框

环境要求

  • Windows 10 及以上（x64）
  • .NET Framework 4.7.2 或更高
  • WebView2 运行时（多数 Win10/11 已预装 Edge 内核）
  • Quicker 1.45.0 及以上

常见问题

  • 窗口空白
    确认已安装 WebView2；若刚更新子程序，可重启 Quicker 后再试
  • 样式异常
    请使用标准 Markdown / GFM 语法；复杂 HTML 标签不在支持范围内
  • 按钮无响应
    检查按钮定义每行格式是否为「标题|Data」，并与 MessageBox3md 自定义按钮写法保持一致
