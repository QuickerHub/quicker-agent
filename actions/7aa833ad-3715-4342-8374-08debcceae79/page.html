捷径面板 n10

把收藏夹里的文本、链接、文件、文件夹与 Quicker 动作，以可自定义布局的超级面板形式快速唤出；单击即可粘贴或执行，尽量不抢占当前窗口焦点。

主要功能

  • 收藏即面板：每个「面板根」对应一组收藏；支持多面板根，左侧树形导航切换
  • 文件夹层级：收藏夹可嵌套；进入子文件夹浏览，支持返回上一级
  • 多种布局：大图标、条带、紧凑列表等样式；窗口尺寸变化时可自动选用更合适的布局
  • 一键执行：单击条目粘贴文本、打开链接/路径，或运行 Quicker 动作（依条目类型而定）
  • 多选批处理：Ctrl / Shift 多选后，可批量粘贴、复制或移除
  • 拖拽排序：在同一层内拖动条目调整顺序
  • 右键维护：空白处可添加文本、批量文本、链接、文件/文件夹等；条目上可重命名、进入子夹、粘贴/复制/移除
  • 键盘导航：方向键移动焦点，Enter 执行当前项；Home / End 跳到首尾
  • 低打扰唤出：面板显示时尽量不抢走前台焦点，便于边浏览边选择

界面说明

  • 单面板：仅一个收藏根时，主区域直接展示该面板下的条目与文件夹
  • 多面板总览：配置了多个面板根时，左侧为可展开的树形导航，右侧为当前面板内容
  • 可拖曳分栏：总览模式下左侧导航宽度可拖动，宽度会保存在本地
  • 置顶：可将面板窗口置顶，避免被其它窗口遮挡（具体以动作设置为准）

界面由 IntelliTools 网页前端渲染；与剪贴板 n10 共用 ClipHost 本地服务。

使用说明

  1. 安装动作后，在 Quicker 中触发即可打开对应超级面板（首次使用需确保剪贴板 n10 / ClipHost 已正常启动）
  2. 在设置 → 面板中配置默认面板、外观与行为（如双击空白关闭等）
  3. 在收藏中整理条目；面板展示的内容与收藏结构同步
  4. 单击条目执行；文件夹双击或菜单「进入收藏夹」进入下一层
  5. 在空白处右键可从剪贴板添加、添加文本/链接/文件，或按提示从 Quicker 面板拖拽动作加入
  6. 再次触发同一面板动作，或执行 panel toggle，可显示/隐藏窗口
打开「自动运行」设置（若需开机自启，请将相关 IntelliTools 动作加入自动运行）

小技巧

  • 多选后于右键菜单批量粘贴，适合连续插入多段文本或路径
  • 长列表面板可用方向键快速定位，再按 Enter 确认
  • 自动化场景可使用 panel show -i <id>、panel hide -i <id> 等命令（id 可为新版 GUID 或旧版数字映射）

环境要求

  • Windows 10 及以上
  • Quicker 1.33.30 及以上，建议 1.45.0+
  • 需安装 .NET 10 共享运行时（与剪贴板 n10 相同）：Windows Desktop Runtime 10.x 与 ASP.NET Core Runtime 10.x 两项均需安装
  • 建议同时安装 剪贴板 n10 并保持 ClipHost 服务可用

运行时安装方式与剪贴板 n10 说明一致，可在剪贴板动作页查看直链与 winget 命令，或通过面板/剪贴板内的 .NET 10 安装引导检测。

常见问题

  • 面板空白或无法连接？
    请先确认 ClipHost 已启动（通常需先运行剪贴板 n10 或相关自动运行动作），并检查 .NET 10 两项运行时是否均已安装。
  • 找不到某个收藏？
    确认该条目属于当前面板根对应的收藏树；多面板时请在左侧切换到正确的面板根。
  • 如何添加 Quicker 动作？
    在面板空白处右键查看提示，从 Quicker 面板将动作按钮拖入即可（与收藏夹添加动作方式一致）。
  • 与旧版「超级面板」的关系？
    本版为 n10 重构，界面与命令 panel 与 IntelliTools 前端一致；旧版数字 id 仍可通过命令映射打开。
