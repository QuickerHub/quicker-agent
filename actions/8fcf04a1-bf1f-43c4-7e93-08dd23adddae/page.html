文本对比

基于 Monaco Editor（VS Code 同款内核）的双栏文本 / 代码对比工具，界面清爽、操作顺手，适合日常 diff 与代码审阅。

核心能力

  • 专业编辑体验：Monaco 内核，支持 VS Code 常用快捷键与专业级编辑手感
  • 灵活输入：可从剪贴板历史取文对比、分步引导输入，或打开空白窗口自由粘贴编辑
  • 语法高亮：内置多种语言，可切换高亮方案；支持自定义悬浮选择面板
  • 阅读友好：自动换行、滚轮缩放、左右栏宽度可调；选中时实时显示字符数
  • 布局自由：面板可拖拽，适配不同屏幕与对比习惯

适用场景

代码审查 · 文档比对 · 配置 / 版本 diff · 教学演示 · 从剪贴板快速核对两段文本

使用说明

  1. 在 Quicker 面板安装动作后，用快捷键或面板按钮触发
  2. 按引导选择「从剪贴板取对比文本」或「手动输入 / 空白编辑」
  3. 在左右栏粘贴或编辑内容，差异高亮会自动更新
  4. 需要时切换语言高亮、自动换行与缩放；遇到问题可在本页「讨论」区留言

安装要求

  • 最低 Quicker 版本：1.38.15，建议使用 1.44.54 及以上
  • 适用于 Windows，通用 / 文本处理场景

使用提示

  • 长文本建议开启自动换行，阅读更轻松
  • 可用滚轮 + Ctrl 缩放字号，细对比更方便
  • 若无法编辑或对比异常，先更新 Quicker 与动作到最新修订版，再在讨论区附上截图
