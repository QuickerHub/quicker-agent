剪贴板 n10

基于 .NET 10 重构的预览版，在保留剪贴板核心能力的同时，优化唤出体验与键盘操作。

本版亮点

  • 类 Win+V 体验：唤出面板时尽量不抢占当前窗口焦点，边浏览边选粘贴更顺手
  • 方向键导航：列表内可用方向键快速移动、选中条目
  • 快速搜索：按 F 聚焦搜索框，在海量记录里迅速定位
  • 持续更新：ClipHost 本地服务、网络子程序兼容等能力在测试版中优先迭代

安装与要求

  • 最低 Quicker 版本：1.39.0，建议使用 1.45.0 及以上
  • 安装后可在面板中触发；建议与稳定版二选一，避免同时常驻冲突
  • 若需开机自启，请在 Quicker「自动运行」中加入本动作
打开「自动运行」设置

.NET 10 运行时（ClipHost 必需）

剪贴板后台服务 ClipHost 基于 .NET 10，且同时使用 WPF 与 ASP.NET Core。 请安装 Windows Desktop Runtime 10.x 与 ASP.NET Core Runtime 10.x 两项共享运行时； 仅安装其中之一时，进程仍可能无法启动。

与 IntelliTools 内「.NET 10 安装引导」一致；版本号随官方更新，下列直链以 10.0.8 为例，其它版本请在 .NET 10 官方下载页 选择对应架构。

手动下载（直链安装包）

运行时 Windows x64 Windows arm64
ASP.NET Core Runtime aspnetcore-runtime-10.0.8-win-x64.exe aspnetcore-runtime-10.0.8-win-arm64.exe
Windows Desktop Runtime windowsdesktop-runtime-10.0.8-win-x64.exe windowsdesktop-runtime-10.0.8-win-arm64.exe

命令行安装（winget）

在提升权限的命令提示符或 PowerShell 中执行（与 IntelliTools 自动安装相同）：

  • winget install Microsoft.DotNet.AspNetCore.10 --accept-package-agreements --accept-source-agreements
  • winget install Microsoft.DotNet.DesktopRuntime.10 --accept-package-agreements --accept-source-agreements

若 winget 不可用，请使用上表直链或官方下载页手动安装。

使用提示

  • 再次触发动作可关闭面板（适合绑定快捷键）
  • 遇到问题可先查看动作页「讨论」区，或到 QQ 群反馈
  • 右键动作按钮 → 设置，可调整界面与行为（与稳定版类似）
  • 面板内可通过相关入口打开 .NET 10 安装引导（检测是否已安装上述运行时）
