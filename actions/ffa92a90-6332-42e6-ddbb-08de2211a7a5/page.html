贴边隐藏

将当前前台窗口贴靠屏幕边缘并自动收起，需要时鼠标移入或获得焦点即可唤出，节省桌面空间又不打断当前操作。

主要功能

  • 四向贴边：支持上 / 下 / 左 / 右贴靠屏幕边缘隐藏
  • 智能暂留：窗口有焦点时保持展开；鼠标悬停在窗口上时同样暂留，避免误收起
  • 延迟显示：可配置唤出延迟，减少贴边弹出过于灵敏、晃眼的情况
  • 统一管理：管理窗口查看已贴边列表，支持右键菜单「取消所有贴边」
  • 子程序调用：可在组合动作中指定贴边方向或批量取消，便于与其它自动化串联

使用说明

  1. 将要隐藏的窗口置于前台（当前活动窗口）
  2. 触发本动作，按提示选择上 / 下 / 左 / 右贴边，或进入管理界面
  3. 窗口贴边收起后，将鼠标移到屏幕对应边缘或窗口露出区域即可唤出
  4. 需要恢复全部窗口时，在管理界面或右键菜单使用「取消所有贴边」，或运行 stopAll 命令
  5. 若感觉弹出过快，可在动作设置中调整延迟显示相关选项

子程序与命令参数

子程序：Cea_窗口贴边隐藏。传入命令字符串（英文，请按下表拼写）：

命令说明
top上贴边
bottom下贴边
left左贴边
right右贴边
manage打开管理窗口
stopAll取消所有贴边，恢复窗口位置

适用场景

聊天 / 笔记 / 监控小窗贴边 · 多显示器边缘收纳 · 临时隐藏前台工具窗 · 与组合动作联动批量贴边或还原

环境要求

  • Windows 10 及以上
  • 适用于通用 Windows 窗口（部分全屏或特殊框架窗口可能不支持）
  • Quicker 1.44.54 及以上（建议使用最新版）

常见问题

  • 退出 Quicker 后窗口找不到了？
    重启 Quicker 后再次运行本动作（任意操作一次即可），退出前处于贴边状态的窗口会恢复。
  • 贴边后点击无反应？
    部分应用内嵌控件（如微信表情面板）可能与贴边交互冲突，可先取消贴边或换用其它边缘。
  • 唤出太灵敏、晃眼？
    在动作设置中增大延迟显示；也可在讨论区反馈你的使用场景。
  • 「取消所有贴边」无效？
    请更新到最新修订版（rev.13+ 已支持右键菜单与 stopAll）；仍异常请在讨论区附上 Quicker 版本与复现步骤。
